# Burpless [![AppVeyor](https://img.shields.io/appveyor/ci/burpless/burpless.svg)](https://ci.appveyor.com/project/burpless/burpless)
Burpless is a .NET implementation of the [Gherkin language](https://docs.cucumber.io/gherkin/) for Behavior Driven Development (BDD). It is fully compatible with .NET Core and .NET Framework.

For complete documentation, please visit the [wiki](https://github.com/burpless/burpless/wiki).

## What does it do
Burpless has the following components:

1. A lexer and parser for the [Gherkin language](https://docs.cucumber.io/gherkin/), including a fully-compiled syntax tree
2. A test adapter, used for Visual Studio and .NET Core projects
3. A [ReSharper](https://www.jetbrains.com/resharper/) plugin to detect tests when using ReSharper
4. A set of [code analyzers](https://docs.microsoft.com/en-gb/visualstudio/extensibility/getting-started-with-roslyn-analyzers) to detect common code mistakes
